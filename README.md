# beerbot-jenkins [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]
> Beerbot plugin for Jenkins CI


## Install

```sh
$ npm install --save beerbot-jenkins
```


## Usage

```js
var beerbotJenkins = require('beerbot-jenkins');

beerbotJenkins('Rainbow');
```
